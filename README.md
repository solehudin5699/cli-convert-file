## Installation

1. Clone this Repository

```sh
  https://github.com/wildandhya/cli-convert-file.git
```

2. Install NPM Packages

```sh
  npm install
```

3. Run the App

```sh
npm install -g
```

## Usage

1. example

```sh
  conv E:\Hello.txt -t json -o E:\helo\Hello.txt
```

2. option

```sh
  -t <type> json | text
```
```sh
  -o <new directory> E:\helo\Hello.txt
```
